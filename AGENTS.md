# Sobre o as cenas do projeto de modo geral
Sempre faça as cenas tendo em vista que o jogo poderá ser rejogado, ou seja, reinicie os estados das cenas, se houver troca de bgs, sempre comece do primeiro quando a cena carregar, etc.
